Download

Install ProxyNova on Windows or build it yourself.

The recommended path is the latest GitHub release. Developers can build the same app from source with Go, Node.js, Wails, and the GeoLite2 database.

Prebuilt app

Windows desktop binary

The release build is a portable Windows executable. Download it from GitHub Releases, place it in a directory you control, and run it normally.

  • Runs locally as a desktop app.
  • Does not require an account or hosted API.
  • Stores app configuration on the local machine.

Build from source

Compile the app locally.

Use this path when you want to audit, modify, or reproduce the build.

Requirements

  • Go 1.22 or newer
  • Node.js 18 or newer
  • Wails v2 CLI
  • GeoLite2-City.mmdb at files/GeoLite2-City.mmdb

Commands

git clone https://github.com/savxzthc/ProxyNova.git
cd ProxyNova
wails build

The Windows output is written to build/bin/ProxyNova.exe.
