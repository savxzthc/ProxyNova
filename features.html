Features

Everything needed to turn messy proxy lists into usable exports.

ProxyNova combines a concurrent Go checker, source scraping, persistent local configuration, and a desktop UI designed for scanning large result sets.

01

Multi-protocol checking

Validate HTTP, HTTPS, SOCKS4, and SOCKS5 entries in parallel. Tune concurrency and timeout values to match your network and the quality of the source list.

02

Built-in scraper

Aggregate proxy entries from configured public sources, remove duplicates, and send the cleaned queue directly into the checker.

03

Anonymity grading

Judge responses are analyzed for header and origin leaks, classifying proxies as Elite, Anonymous, Transparent, or failed.

04

Offline geography

When GeoLite2-City.mmdb is bundled, ProxyNova resolves country and city locally. That keeps lookups fast and avoids sending addresses to a remote geo API.

05

Large result handling

The React UI uses virtualized tables so very large result sets remain searchable, sortable, and scrollable without freezing the window.

06

Export-ready filters

Filter by protocol, status, latency, anonymity, and location, then export working lists as TXT, CSV, or JSON for downstream tooling.

Local-first architecture

Settings and results stay on your machine.

ProxyNova stores configuration in an embedded bbolt database. Source lists, judge endpoints, checker settings, and export preferences persist across launches without a cloud account.
