The fast path

Raw lists in. Clean results out.

ProxyNova is built for high-volume validation work: import lists, scrape configured sources, test across protocols, grade anonymity, resolve geography locally, then export only the results you need.

01

Collect

Paste IP:PORT entries, import local files, or run the built-in scraper across configured public sources.

02

Validate

Run concurrent checks with timeout, retry, protocol, judge, and anonymity settings tuned to your network.

03

Export

Filter by working status, latency, protocol, anonymity, and location, then export TXT, CSV, or JSON.

Designed for real queues

Desktop speed without a hosted backend.

ProxyNova runs as a native Wails desktop app with a Go checking engine and a React interface. The app keeps settings, source lists, judges, and results on your machine.

  • Configurable concurrency and timeout controls for different networks.
  • Offline MaxMind GeoLite2 country and city lookup when the database is bundled.
  • Virtualized result tables built to stay responsive with very large lists.

Privacy by design

No account. No telemetry. No remote dashboard.

ProxyNova is a local tool. Your proxy lists, checker settings, judge URLs, scraper sources, and exports stay on the computer running the app.

Privacy details

Built for legitimate testing

A checker for authorized proxy validation.

Use ProxyNova for development, research, list hygiene, and network administration. Do not use it for spam, credential attacks, access-control evasion, or abusive automation.
