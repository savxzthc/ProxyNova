Privacy

ProxyNova is local-first by design.

The app is built for users who want to inspect and process proxy lists without uploading them to someone else's service.

No accounts or hosted dashboard

ProxyNova does not require a user account. The website links to downloads, documentation, and source code. The desktop app runs locally.

No telemetry claim for the published app

The project is described as a local desktop checker with no phoning home. Users should still review the open-source code and release artifacts if they require high assurance before running any network tool.

Local data

The app stores settings, source configuration, judges, and result state in local application data. Exports are written only when you choose to save or copy results.

Network traffic created by the app

Checking proxies necessarily creates outbound network traffic to the judge endpoints and through the proxies being tested. Scraping source lists creates outbound requests to the configured source URLs. These requests are part of the app's core function.

Website data

This static website has no contact form, analytics script, tracking pixel, or third-party embed. External navigation happens only when you choose links to GitHub or release downloads.
